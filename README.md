## Gallery
This is my personal gallery where I put pictures that I find interesting. it is made with html, css and javascript.

You can access the gallery [here](https://gallery-s.netlify.app/).

# Local Development
- Clone the repository.
- Open `index.html` in your browser or just use a live server extension.
- Tweak the code as you like :)
